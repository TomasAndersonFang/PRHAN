# PRHAN
A hybrid attention based neural model for pull request description generation

## Dependency
Successfully tested in Ubuntu 18.04
- Python == 3.7
- PyTorch == 1.6.0
- tqdm == 4.48.2
- torchtext == 0.6.0
- spacy == 2.3.2
- numpy == 1.18.5
